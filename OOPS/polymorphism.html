Polymorphism means one name more than one form.

Types of Polymorphism:
  • Compile Time Polymorphism (Method Overloading)
  • Run Time Polymorphism (Method Overriding)

    • Compile Time Polymorphism (Method Overloading):

    Not supported by Python.


    Run Time Polymorphism (Method Overriding):

    Same method signature used in parent and child class.

    Method overriding can be achieved by two ways:

  • using class name


  • using super() method
