Inheritance means code of re-usability.

Types of Inheritance:
  • Single Inheritance
  • Multilevel Inheritance
  • Multiple Inheritance
  • Hierarchical Inheritance
  • Hybrid Inheritance

    • Single Inheritance:


    Class B is inheriting from class A, so we can access members of class A using class B object.


    Multilevel Inheritance:

    There is more than two classes involved using level by level (C derived from B, B derived from A).



    Class C is from inheriting class B and class B is from inheriting class A, so we can access members of class A and B using class C object.


    Multiple Inheritance:

    class derived from more than one parent class (C derived from A and B both).



    Class C is inheriting from class A and class B, so we can access members of class A and B using class C object.


    Hierarchical Inheritance:

    More than one derived class are created from a single base


    Hybrid Inheritance:

    This form combines more than one form of inheritance. Basically, it is a blend of more than one type of inheritance
