Array 1D :

Back

Create array : arr = [0]*size => arr = [0]*4

Update array element : arr[2] = 7

Get array element : element = arr[2]

Get length of array : len(arr)

Sort array in ascending order : arr.sort()

Sort array in descending order : arr.sort(reverse=True)

Sort array based on specific comparator : arr.sort(key=lambda a:a[1]) => based on second value

Sort based on first value and if both pair first values are equal then sort based on second value:
 list = [(2,4), (2,3), (1,8)]
list.sort(key=lambda a: a[1] if a[0] == a[1] else a)
output => [(1, 8), (2, 3), (2, 4)]

Creating array without element initialize: arr = [ ]*size => arr = [ ], it is empty list, accessing any element will throw exception
