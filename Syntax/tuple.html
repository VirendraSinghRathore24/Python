Tuples are ordered, unchangeable, Immutable, readonly.

Once tuple declared, it can not modify after that.

Create tuple : t = ("Viren", "Sanju", "Rudransh")

Usually we use small brackets for tuple, but it is not mandatory : t = "Viren", "Sanju", "Rudransh"

As python does not depend on data type so tuple can contain mixed type as well : t = ("Viren", 5, 2.22, True)

Get tuple items : t[0]

Tuple allows duplicate values as well
